Este é um projeto que vista facilitar o registro de ordens de serviço para manutenção de ar-condicionados.

Os usuários registrados podem fazer login, ver suas ordens de serviço e registrar novas ordens.

Cada ordem contém numero, local, descrição, fotos iniciais e ambientes. Os ambientes são o equipamento e manutenção efetuada. A manutenção pode ser preventiva ou corretiva, possui a descrição do equipamento, conserto e as fotos de evidência. Ao finalizar o serviço, o técnico pode gerar um pdf com o relatório final e exportar/compartilhar.
